# `042` printUserBadgeVoid

## 📝 Instructions:

1. Write a procedure called `printUserBadge`. Given a user's name and level, `printUserBadge` prints a formatted badge to the console.

2. The badge must follow this format: `[Level {level}] {name}` (for example, `[Level 2] Ana`).

3. This is a **void** procedure: it must not return any value. Use `console.log()` to print the badge.

## 📎 Example:

```ts
printUserBadge('Ana', 2);
// console output --> [Level 2] Ana
```

```ts
printUserBadge('Carlos', 5);
// console output --> [Level 5] Carlos
```

## 💡 Hint:

+ A void function performs an action (such as printing) but does not return a value.
+ Do not use `return` with a value inside this function.
